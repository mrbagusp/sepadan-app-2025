# Sepadan - A Christian Dating App

Sepadan is a Flutter-based mobile application designed to help Christians connect and find meaningful relationships. This app provides a safe and welcoming environment for users to create profiles, browse potential matches, and communicate with others who share their faith and values.

## Features

*   **Authentication:** Secure sign-up and sign-in with email/password and Google Sign-In.
*   **User Profiles:** Create and customize user profiles with personal information, photos, and faith-based details.
*   **Matching:** A smart matching algorithm that suggests compatible partners based on user preferences and profile information.
*   **Real-time Chat:** Communicate with matches in real-time through a secure and private messaging system.
*   **Theming:** A dynamic theme that supports both light and dark modes.

## Tech Stack

*   **Frontend:** Flutter
*   **Backend:** Firebase (Authentication, Firestore, Storage)
*   **State Management:** Provider
*   **Routing:** go_router

## Getting Started

### Prerequisites

*   Flutter SDK
*   Firebase CLI

### Installation

1.  Clone the repository:

    ```bash
    git clone https://github.com/your-username/sepadan-app.git
    ```

2.  Install dependencies:

    ```bash
    flutter pub get
    ```

3.  Run the app:

    ```bash
    flutter run
    ```
